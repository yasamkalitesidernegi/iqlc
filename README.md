# IQLC 2026 Web Sitesi

Bu paket, GitHub Pages için hazır statik konferans sitesi içerir.

## Dosyalar
- `index.html`
- `styles.css`
- `script.js`

## GitHub Pages yayına alma
1. Repo içindeki mevcut dosyaları açın.
2. Bu paketteki dosyaları repo kök dizinine yükleyin.
3. GitHub > Settings > Pages bölümünden ana branch'i seçin.
4. Site birkaç dakika içinde yayına alınır.

## Özellikler
- Türkçe / İngilizce dil değiştirici
- Tek sayfa konferans yapısı
- Mobil uyumlu tasarım
- Geri sayım alanı
- Referans siteden esinlenen kurumsal görünüm

## Not
- `Bildiri Gönder` butonu ve `Şablon Dosyası` bağlantısı şu an genel placeholder bağlantı olarak bırakıldı. Bunları gerçek form ve şablon linkleriyle güncelleyin.
- Görseller ve logolar hazır olduğunda `index.html` içine kolayca eklenebilir.
